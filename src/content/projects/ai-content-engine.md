I wanted to post consistently on LinkedIn without it eating my week — so I built a system to do it.

The engine drafts posts in my own voice and structure, generates matching on-brand visuals,
runs every draft through an automated confidentiality check, and schedules the approved ones —
all orchestrated with Claude Code and a set of small Python scripts.

It’s the clearest example of how I work today: spot a repetitive, data-shaped problem, and use
AI to turn it into a lightweight tool that actually ships and keeps running on its own.
